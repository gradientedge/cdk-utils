CloudflareZoneManager

CloudflareZoneManager

Provides operations on Cloudflare Zone using Pulumi

  • A new instance of this class is injected into CommonCloudflareConstruct constructor.
  • If a custom construct extends CommonCloudflareConstruct, an instance is available within the context.

Constructor

new CloudflareZoneManager()

Description:
Source:
  • cloudflare/services/zone/main.ts, line 21
Example
```
import { CommonCloudflareConstruct } from '@gradientedge/cdk-utils'

class CustomConstruct extends CommonCloudflareConstruct {
  constructor(parent: pulumi.ComponentResource, id: string, props: CommonCloudflareStackProps) {
    super(parent, id, props)
    this.props = props
    this.zoneManager.createZone('MyZone', this, props)
  }
}
```
