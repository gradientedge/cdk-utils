## Installation
### ![cmd]
With **npm**:
```shell
npm install --save @gradientedge/common-aws
```

With **yarn**:
```shell
yarn add @gradientedge/common-aws
```

### ![purescript]
```
"@gradientedge/common-aws": "latest"
```
