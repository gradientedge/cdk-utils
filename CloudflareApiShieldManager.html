CloudflareApiShieldManager

CloudflareApiShieldManager

Provides operations on Cloudflare Api Shield

  • A new instance of this class is injected into CommonCloudflareConstruct constructor.
  • If a custom construct extends CommonCloudflareConstruct, an instance is available within the context.

Constructor

new CloudflareApiShieldManager()

Description:
Source:

Example
```
import { CommonCloudflareConstruct, CommonCloudflareConstruct } from '@gradientedge/cdk-utils'

class CustomConstruct extends CommonCloudflareConstruct {
  constructor(parent: Construct, id: string, props: CommonCloudflareStackProps) {
    super(parent, id, props)
    this.props = props
    this.apiShieldManager.createApiShield('MyApiShield', this, props)
  }
}
```
